# Changes

## Version 0.2.1 (2022-05-09)

- Added Bulk subject definition support ([#3](https://github.com/geo-knowledge-hub/geo-deposit-react/issues/3) and [#14](https://github.com/geo-knowledge-hub/geo-deposit-react/issues/14))
- Package renamed to `@geo-knowledge-hub/geo-deposit-react` using [scoped pattern](https://docs.npmjs.com/cli/v8/using-npm/scope)

## Version 0.2.0 (2022-04-24)

- Custom `DepositRecordSerializer` class (based on [react-invenio-forms](https://github.com/inveniosoftware/react-invenio-forms)) to support the GEO Knowledge Hub API Data Model;
- [Formik](https://formik.org/docs/overview) fields based on [react-invenio-forms](https://github.com/inveniosoftware/react-invenio-forms) to support the custom GEO Knowledge Hub fields.
  - Target Audience field;
  - Engagement Priorities field;
  - GEO Work Programme Activities field.
